Administration guide

First connection

1. Modify your email and password
The first time you connect to the application, an admin account is provided :
  • Login : admin
  • Password : admin
Go to the profile panel, clicking the profile icon on the top left of the screen, and update the information.

Site and Rooms

2. Setup your data
Initialize your sites and related rooms with the GUI provided.

Users

3. Create users
Create users with their according role. On creation, each user will receive an e-mail to be notified of their password.
  • Role Admin : Access the whole site
  • Role Supervisor : Manage Feedbacks. Cannot administer RooMeeting
  • Role User : Only booking feature is allowed
