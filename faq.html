  1. Where can I install RooMeeting ?
  2. What are the server requirements ?
Where can I install RooMeeting ?
RooMeeting is a client server application. The client is represented by any browser available on the web.



The server engine is the only step in order to make it work.

It can be deployed on a Linux© server as well as on your Windows© Desktop. All depends on your IT and requirements.
[top]
What are the server requirements ?
A Java Virtual Machine (JVM) version 6+ have to be installed in the machine the installation is processed on.

It is often included in a Windows© installation.



To get a JVM that fit to your machine, visit http://www.java.com/fr/download/ and install on your server machine only.
